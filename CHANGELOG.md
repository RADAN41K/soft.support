# Changelog

## v1.0.6
- Установник видаляє старий ключ автозапуску (launcher.vbs) з реєстру
- USB: виключено Bluetooth, Wi-Fi, клавіатури та миші зі списку пристроїв

## v1.0.5
- Видалено клікабельне посилання на Telegram (залишено текст як інформаційний напис)
- Кнопки "Редагувати", "Логи", "Вихiд" розташовані в стовпчик
- Додано кнопку "Вихiд" для швидкого згортання вікна
- Контекстне меню (правий клік) у полі введення коду точки
- Повторний запуск з ярлика показує вікно, якщо програма вже працює в треї
- Виправлено помилку SSL на Windows (pip-system-certs + certifi + truststore fallback)
- QR-код збільшено до 190px, номер точки та назва на окремих рядках
- COM порти: статус підключення (зайнятий/знайдено/вільний/відключено)
- COM порти: показано hardware ID для діагностики
- USB порти: перехід з PowerShell на Python WMI бібліотеку (стабільніше, без мерцання)
- Виправлено подвійне DPI-масштабування вікна на Windows
- Програма запускається згорнутою в трей
- Контекстне меню (правий клік) у списку портів: копіювати виділене або все

## v1.0.4
- Надійне автооновлення: установник чекає завершення програми перед оновленням
- Примусове закриття програми при видаленні/оновленні через Inno Setup
- Версія читається з файлу VERSION напряму (Inno Setup, PyInstaller, Python)
- Версія відображається в заголовку вікна

## v1.0.3
- Тихе автооновлення через Inno Setup (`/VERYSILENT`)
- Єдине джерело версії — файл `VERSION`
- Встановлення в `AppData` (не потрібні права адміністратора)
- Автозапуск при завантаженні системи (чекбокс + установник)
- Логи перенесено в системні директорії, перегляд через пароль
- Виправлено Ctrl+V на нелатинських розкладках (Windows/macOS)
- Приховано консольні вікна subprocess на Windows

- Виправлено визначення USB-портів (фізичні номери портів на Windows)

## v1.0.0
- Перший реліз
