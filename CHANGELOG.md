# Changelog

## v1.0.8
- Автооновлення: окремі збірки для Windows 7, Windows 10, Windows 11
- Пiдтримка Windows 7: Python 3.8, окремi залежностi (requirements-win7.txt)
- Виправлено введення букв c/v у полi токена на Windows 10 (конфлiкт Ctrl з NumLock)
- Збiрка Linux через Docker (./build-linux.sh) з macOS/Apple Silicon
- .deb пакет для Ubuntu 22.04+ з залежностями та ярликом на робочому столi
- Linux: вiкно показується одразу замiсть згортання в трей (трей не пiдтримується на Ubuntu 24)
- Linux: кнопка закриття мiнiмiзує вiкно в таскбар замiсть закриття додатку
- Linux: iконка в системному треї через Ayatana AppIndicator
- Linux: автоматичне включення GNOME-розширення appindicatorsupport при установцi
- Linux: додаток доступний в пошуку та можна закрiпити в панелi задач
- Docker: повторна збiрка без помилки iснуючого контейнера
- .deb: установка через apt install (автоматичне пiдтягування залежностей)
- macOS: iконка в menu bar з меню (показати/логи/закрити), без Dock-iконки

## v1.0.7
- Індикатор сканування портів (зелена - готово, оранжева - сканування, червона - помилка)
- Оптимізовано WMI запит: тільки USB пристрої замість усіх PnP (менше навантаження на CPU)
- USB: фільтрація за класом пристрою Windows замість імені (надійніше визначає периферію)
- Замінено polling кожні 3с на event-driven моніторинг пристроїв та мережі
- Роздільне сканування: подія USB - тільки порти, подія мережі - тільки IP
- macOS: миттєве виявлення USB через IOKit (ctypes)
- Linux: watchdog на /dev + netlink socket для мережі
- Windows: WMI event subscription + NotifyAddrChange для мережі
